# Rails React With Webpack

This repo is a demonstration of how to use Rails and React as one app using Webpack.

## Built With

- Major languages (Ruby)
- Framworks (Ruby on Rails)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- A text editor(preferably Visual Studio Code)

### Install

- Ruby
- Ruby on Rails
- PostgresSQL
- Rspec
- React-router

### Using it Locally

- Clone the repository using the below commands.

- `https://github.com/Yazino12/rails-react-webpack.git `
- `cd rails-react-webpack`
- `rails s`

### Setup

Install gems with:

```
bundle install
```

Setup database with:

```
rails db:create
rails db:migrate
```

### Usage

Start server with:

```
rails server
```

Visit http://localhost:3000/ in your browser.
